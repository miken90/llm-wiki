# LLM Wiki

You have access to a shared knowledge wiki via the qmd MCP server.

- **WIKI_ROOT**: {{WIKI_ROOT}}
- **Schema**: Read {{WIKI_ROOT}}/wiki-schema.md for all conventions

## When to use

- Before non-trivial work → search wiki for prior knowledge
- When discovering durable knowledge → write back to wiki

## Workflow

1. Use qmd MCP tools to search wiki
2. Read matched pages for context
3. To write: follow wiki-schema.md conventions
4. Search before creating — update existing pages, don't duplicate
5. Commit wiki changes separately from project changes
